# Node-RED 动态象棋谱库 项目计划

## 项目概述
在 Node-RED 中实现一个可交互的动态象棋谱展示组件，支持：
- 加载标准棋谱（PGN格式）和 FEN 局面
- 分步演示开局/中局/残局
- 支持用户交互走棋
- 可嵌入到 Node-RED 仪表盘或网页中

## 目标
1. 开发自定义 Node-RED 节点，支持动态渲染象棋棋盘
2. 支持棋谱导入、分步播放、手动走棋交互
3. 整理技术协议和开发文档，方便后续维护和扩展
